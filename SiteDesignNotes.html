Interactive AngularJS Directives

Copy to Clipboard

Usage
<copy-to-clipboard-button target="id of target element" success-message="text" class="class name"></copy-to-clipboard-button>
Attribute Required Description
target Yes The id attribute value of the target element whose inner text is to be copied to the clipboard when this directive's button is clicked.
success-message No The message to display after text is succssfully copied to clipboard.
class No The class names to apply to the button produced by this directive. Defaults to "btn btn-light btn-outline-dark p1".
Single-line Content
Copy element containing single-line code content to clipboard. <copy-to-clipboard-button target="idOfElementContainingCodeAsInnerText" success-message="Code copied"></copy-to-clipboard-button> <code class="text-monospace" id="idOfElementContainingCodeAsInnerText">hasRole('admin') && !current.IsNewRecord()</code>
Multiple-line Content
Copy element containing mutiple-line code content to clipboard. <copy-to-clipboard-button target="idOfElementContainingCodeAsInnerText" success-message="Code copied"></copy-to-clipboard-button> <code class="pre-scrollable multi-line" id="idOfElementContainingCodeAsInnerText">Multi-Line Text</code>

Navigation Links

Usage
<sn-nav-link path-nodes="delimited/text" href="relativeURL" href-model="modelVar" link-index="number" node-separator="delimiterChar" to-nav="true|false" target="string"></sn-nav-link>
Attribute Required Description
path-nodes Yes Delimited text string representing the path nodes to navigate.
href No The static URL that is relative to the target ServiceNow instance.
href-model No The model variable containing the URL that is relative to the target ServiceNow instance.
link-index No The zero-based index of the navigation node that is to be rendered as a hyperlink. The default is the the last node. This is ignore if no href or href-mode is specified.
node-separator No The character that separates individual navigation node text in path-nodes. The default is a forward slash ("/").
to-nav No Indicates whether the link is encoded as a uri query parameter to the ServiceNow nav_to.do page.
target No The target for the hyperlink. The default is _blank.
Examples
Simple statically defined relative URL.
<sn-nav-link path-nodes="System LDAP/Data Sources" href="/sys_data_source_list.do" /> Will be transpiled as: <samp class="navPath"> <span> <var>System LDAP</var> &rArr; </span> <a href="https://yourinstance.servicenow.com/sys_data_source_list.do" target="_blank"> <var class="targetName">Data Sources</var> </a> </samp>
Navigation path with alternate link index.
<sn-nav-link path-nodes="System LDAP/Transform Maps/Enterprise User Import" link-index="1" href="/sys_transform_map_list.do" /> Will be transpiled as: <samp class="navPath"> <span> <var>System LDAP</var> &rArr; </span> <a href="https://yourinstance.servicenow.com/sys_transform_map_list.do" target="_blank"> <var>Transform Maps</var> </a> <span> &rArr; <var class="targetName">Enterprise User Import</var> </span> </samp>
Navigation path with alternate node separator.
<sn-nav-link path-nodes="Configuration|Identification/Reconciliation|CI Identifiers" node-separator="|" href="/cmdb_identifier_list.do" /> Will be transpiled as: <samp class="navPath"> <span> <var>Configuration</var> &rArr; <var>Identification/Reconciliation</var> &rArr; </span> <a href="https://yourinstance.servicenow.com/cmdb_identifier_list.do" target="_blank"> <var class="targetName">CI Identifiers</var> </a> </samp>
Statically defined relative URL encoded in nav_to.do query parameter.
<sn-nav-link path-nodes="System LDAP/Data Sources" href="/sys_data_source_list.do" to-nav="true" /> Will be transpiled as: <samp class="navPath"> <span> <var>System LDAP</var> &rArr; </span> <a href="https://yourinstance.servicenow.com/nav_to.do?uri=%2Fsys_data_source_list.do" target="_blank"> <var class="targetName">Data Sources</var> </a> </samp>
Bound URL model.
<sn-nav-link path-nodes="System LDAP/Data Sources" href-model="modelVar" /> Will be transpiled as: <samp class="navPath"> <span> <var>System LDAP</var> &rArr; </span> <a href="https://yourinstance.servicenow.com/sys_data_source_list.do" target="_blank"> <var class="targetName">Data Sources</var> </a> </samp>
Navigation path with no relative URL.
<sn-nav-link path-nodes="System Policy/Rules/Caller VIP Lookup Rules" /> Will be transpiled as: <samp class="navPath"> <span> <var>System Policy</var> &rArr; <var>Rules</var> &rArr; </span> <var class="targetName">Caller VIP Lookup Rules</var> </samp>
