fieldsGrid

Purpose: Display name/value pairs where the name is displayed to the left of the value.

Example Code

<table class="table table-striped fieldsGrid"> <thead class="thead-dark"> <tr> <th scope="colgroup" colspan="2">Col 1</th> <th scope="colgroup" colspan="2">Col 2</th> <th scope="colgroup" colspan="2">Col 3</th> </tr> </thead> <tbody> <tr> <th>Field 1:</th> <td>Value 1</td> <th>Field A:</th> <td>Value A</td> <th>Field B:</th> <td>Value B</td> </tr> <tr> <th scope="row">Field 2:</th> <td>Value 2</td> <th>Field C:</th> <td colspan="3">Value C</td> </tr> <tr> <th>Field 3:</th> <td>Value 3</td> <th>Field D:</th> <td>Value D</td> <th>Field E:</th> <td>Value E</td> </tr> </tbody> </table>

Example Display

Field 2: Value 2 Field C: Value C
Field 3: Value 3 Field D: Value D Field E: Value E
