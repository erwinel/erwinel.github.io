Initial Update Set

Purpose

This applies the initial ServiceNow confuration options as follows:
  • Schedule Definitions
    • 0600 - 1800 Every Day Incl Weekend and Holidays
    • 0600 - 1800 Weekdays Incl Holidays
    • 0600 - 1800 Weekdays Excl Holidays
  • Email Template Classification
    Adds CLASSIFICATION: UNCLASSIFIED//FOR OFFICIAL USE ONLY to top and bottom of several email templates.
  • Custom User table fields
    • Black phone
    • Grey Phone
    • Red Phone
    • Rank
    • TDA Position
  • Setup Scripts
    Adds 2 Fix Scripts that will be used during the configuration process.
    1. Initial Army Config
    2. Army Bulk Plugin Activation
