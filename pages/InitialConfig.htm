Overview

  1. Import and apply initial update set.
  2. Import Utility module from source control.
  3. Set SDLC stage and execute initial config fix script.
  4. Execute bulk plugin activation fix script.
  5. Import Group Name Registry module from source control.

Initial Global Update Set

Load and apply the initial global update set.
  1. Navigate to System Update Sets ⇒ Retrieved Update Sets
  2. Click on Import Update Set from XML under Related Links.
  3. Click on the Choose File button and upload the sys_remote_update_set_Initial_Setup.xml File in the downloads folder of this documentation.
  4. Once it is loaded, open the update set and click the Preview Update Set button.
  5. After it is validated, click the Commit Update Set to apply it.

Import Utility Application

Initial Config

  1. Navigate to System Definition ⇛ Fix Scripts.
  2. Open the fix script named Initial Army Config.
  3. Click Run Fix Script.
  4. Click Proceed if you wish to wait for the script to finish; otherwise, click Proceed in Background to let it run in the background. This script can take several minutes to finish.

Bulk Plugin Activation

  1. Navigate to System Definition ⇛ Fix Scripts.
  2. Open the fix script named Army Bulk Plugin Activation.
  3. Click Run Fix Script.
  4. Click Proceed if you wish to wait for the script to finish; otherwise, click Proceed in Background to let it run in the background. This script can take a long time to finish.
