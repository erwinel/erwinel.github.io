ServiceNow Application, Configuration and Development Repositories

Overview

The collection of Github repositories referenced by this documentation contain update sets, source code and documentation for installing and maintaining custom ServiceNow applications as well as for the initial configuration of ServiceNow instances.

See for initial ServiceNow configuration steps. See for information on global update sets. Otherwise, use other menu options for individual application documentation.

Development

There are 2 special repositories, and , which are intended to assist with complex source code development for other applications. These repositories use node.js and NPM modules to compile TypeScript sources and generate JavaScript. This compiled JavaScript must then be manually ported to the actual JavaScript code that will be used in the individual applications.
