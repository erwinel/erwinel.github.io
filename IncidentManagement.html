Priority Assessment

Incidents submitted using the "Submit a New Incident" service catalog item prompt the user to provide information, which is used to calculate the initial Impact and Urgency values for the incident. In turn, the Impact and Urgency values, together with the user's VIP status determine the actual incident property.
The following form uses a copy of the actual JavaScript code from ServiceNow to demonstrate the results of urgency, impact and priority value calculations.

Calculations

Using assumed values
When neither {{ users_impacted.label }} nor {{ productivity_impact.label }} are selected for non-mission-related incidents for non-VIP users, a value of {{ productivity_impact.value }} is assumed. {{ users_impacted.label }} is not specified: Assuming same value as the {{ productivity_impact.label }} selection. {{ productivity_impact.label }} is not specified: Assuming same value as the {{ users_impacted.label }} selection.

Impact

Impact = Math.round(
    (
        ([{{ users_impacted.label }}] + [{{ productivity_impact.label }}])
        ×
        ([{{ is_mission_related.label }} Delta] + [{{ caller_vip.label }} Delta])
    ) / (11.0 / 3.0)
);
⇒     (
        ({{ users_impacted.value }} + {{ productivity_impact.value }}) × ({{ is_mission_related.operand }} + {{ caller_vip.operand }})
      ) / (11.0 / 3.0)
⇒    ({{ users_impacted.value + productivity_impact.value }} × {{ is_mission_related.operand + caller_vip.operand }}) / (11.0 / 3.0)
⇒    {{ impact.value }} = {{ (users_impacted.value + productivity_impact.value) * (is_mission_related.operand + caller_vip.operand) }} / (11.0 / 3.0);
Result: {{ impact.rounded }} = Math.round({{ impact.value }});

Urgency

current.urgency = Math.round(
    (
        [{{ productivity_impact.label }}] + (
            ([{{ caller_vip.label }}] == true) ?
                (([{{ is_mission_related.label }}] == true) ? 1 : 2) :
                (([{{ is_mission_related.label }}] == true) ? 3 : 4)
        )
    ) * 0.35
);
⇒  ( {{ productivity_impact.value }} + (
        ([{{ caller_vip.label }}: {{ caller_vip.value }}] == true) ?
            (([{{ is_mission_related.label }}] == true) ? 1 : 2) :
            (([{{ is_mission_related.label }}] == true) ? 3 : 4)
    ) ) * 0.35
⇒  ( {{ productivity_impact.value }} + (
        ([{{ is_mission_related.label }}: {{ is_mission_related.value == "Yes" }}] == true) ? {{ (caller_vip.value) ? 1 : 3 }} : {{ (caller_vip.value) ? 2 : 4 }}
    ) ) * 0.35
⇒  {{ urgency.value }} = ({{ productivity_impact.value }} + {{ (caller_vip.value) ? ((is_mission_related.value == "Yes") ? 1 : 2) : ((is_mission_related.value == "Yes") ? 3 : 4) }}) * 0.35
Result: {{ urgency.rounded }} = Math.round({{ urgency.value }});
